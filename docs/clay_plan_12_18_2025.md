## Step 1: Create Scrape Table in Clay

### Table Structure

| Column | Type | Purpose |
|--------|------|---------|
| `query` | Text (input) | Search query, e.g., "HVAC companies Columbus Ohio" |
| `outscraper_response` | HTTP API | Calls Outscraper API |
| `businesses` | Expand list | Flattens response to rows |

### HTTP API Column Config

```
Method: GET
URL: https://api.outscraper.cloud/google-maps-search

Query Parameters:
  - query: {{query}}
  - limit: 500
  - async: false

Headers:
  - X-API-KEY: [YOUR_OUTSCRAPER_API_KEY]
```

### Seed Queries (10 rows = ~4-5k leads)

```
HVAC companies Ohio
HVAC companies Michigan
HVAC companies Indiana
HVAC companies Pennsylvania
HVAC companies New York
HVAC companies New Jersey
HVAC companies Kentucky
HVAC companies West Virginia
HVAC companies Illinois
HVAC companies Connecticut
```

### Fields to Extract from Response

| Outscraper Field | Map To | Notes |
|------------------|--------|-------|
| `name` | company | Business name |
| `site` | domain | Website URL |
| `phone` | phone_number | Phone number |
| `reviews` | reviews_count | Review count (for scoring) |
| `city` | city | City |
| `state` | state | State |
| `place_id` | place_id | **Critical for deduping** |
| `full_address` | address | Full address |

## Step 2: Pipe to Enrichment Table

### Enrichment Columns (where Clay credits get burned)

1. **Find Person** - Find owner/decision maker from company
2. **Find Email** - Get work email for that person
3. **Verify Email** - Confirm email is valid

### Credit Estimate

- ~30-40% of leads will already have email from GMB listing (free)
- Remaining 60-70% need enrichment
- ~3-4 credits per enriched lead
- **4,000 leads × 70% × 4 credits = ~11,200 credits**

### Skip Enrichment If:

- Lead already has email from Outscraper (`email` field populated)
- Use Clay conditional: `IF email IS NOT EMPTY, skip enrichment`

## Step 4: Score & Push (Already Built)

### Scoring (runs automatically in Supabase)

```sql
UPDATE leads SET score = (
  CASE WHEN service_software IN ('ServiceTitan', 'Jobber', 'Housecall Pro') THEN 15 ELSE 0 END
  + CASE WHEN reviews_count >= 500 THEN 10 WHEN reviews_count >= 100 THEN 7 WHEN reviews_count >= 25 THEN 4 WHEN reviews_count >= 10 THEN 2 ELSE 0 END
  + CASE WHEN linkedin_url IS NOT NULL AND linkedin_url != '' THEN 3 ELSE 0 END
  + CASE WHEN domain IS NOT NULL AND domain != '' THEN 2 ELSE 0 END
);
```

### Push to Smartlead

```bash
python scripts/smartlead/push_leads_to_smartlead.py
```
